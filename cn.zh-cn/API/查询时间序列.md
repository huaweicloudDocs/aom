# 查询时间序列<a name="ListSeries"></a>

## 功能介绍

该接口用于查询系统当前可监控的时间序列列表，可以指定时间序列命名空间、名称、维度、所属资源的编号（格式为：resType\_resId），分页查询的起始位置和返回的最大记录条数。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=AOM&api=ListSeries)中调试该接口。

## URI

POST /v2/\{project\_id\}/series

## 请求示例

```
示例1（通过namespace+appName+clusterName查询）
/v2/{project_id}/series
{
    "series": [
        {
            "namespace": "PAAS.CONTAINER",
            "dimensions":[
                {
                    "name":"appName",
                    "value":"demo"
                },
                {
                    "name":"clusterName",
                    "value":"test"
                }
            ]
        }
    ]
}

```
