# 修改阈值规则<a name="UpdateAlarmRule"></a>

## 功能介绍

该接口用于修改一条阈值规则。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=AOM&api=UpdateAlarmRule)中调试该接口。

## URI

PUT /v2/\{project\_id\}/alarm-rules
