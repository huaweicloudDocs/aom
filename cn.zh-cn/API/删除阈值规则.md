# 删除阈值规则<a name="DeleteAlarmRule"></a>

## 功能介绍

该接口用于删除阈值规则。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=AOM&api=DeleteAlarmRule)中调试该接口。

## URI

DELETE /v2/\{project\_id\}/alarm-rules/\{alarm\_rule\_id\}
